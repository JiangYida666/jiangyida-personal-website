## 部署到 GitHub Pages 步骤

### 第一步：创建 GitHub 仓库

1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 填写仓库信息：
   - Repository name: `yida-personal-website` (或其他名称)
   - Description: "蒋奕达的个人学术网站"
   - 选择 Public
   - 勾选 "Add a README file"

### 第二步：上传代码到 GitHub

1. 在本地打开 Git Bash 或终端
2. 执行以下命令：

```bash
# 进入项目目录
cd "e:\高老师作业\个人网站"

# 初始化 Git 仓库
git init
git add .
git commit -m "初始提交：个人Quarto网站"

# 连接到 GitHub 仓库
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

### 第三步：启用 GitHub Pages

1. 进入 GitHub 仓库页面
2. 点击 "Settings" 标签
3. 左侧菜单选择 "Pages"
4. 在 "Build and deployment" 部分：
   - Source: "GitHub Actions"
5. 系统会自动检测到我们的发布配置

### 第四步：等待部署完成

1. 点击仓库顶部的 "Actions" 标签
2. 查看 "Publish Quarto Site" 工作流状态
3. 等待约 2-5 分钟，直到显示绿色勾号 ✅
4. 网站将发布到：`https://你的用户名.github.io/仓库名`

## 本地预览（可选）

如果您想在本地预览网站：

1. 安装 Quarto：https://quarto.org/docs/get-started/
2. 在项目目录运行：
   ```bash
   quarto preview
   ```
3. 打开浏览器访问：http://localhost:4200
