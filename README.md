# 蒋奕达个人网站

这是一个使用 Quarto 构建的个人学术网站。

## 网站结构

- `index.qmd` - 首页
- `about.qmd` - 关于页面
- `_quarto.yml` - 网站配置文件
- `styles.css` - 自定义样式
- `.github/workflows/deploy.yml` - GitHub Actions 部署工作流

## 部署

网站通过 GitHub Actions 自动部署到 GitHub Pages。

## 技术栈

- **Quarto** - 文档发布系统
- **Markdown** - 内容编写
- **CSS** - 样式定制
- **GitHub Actions** - 自动部署
